OVERVIEW

This module provide two fields Is a Customer, Is a Vendor in to the partner to identify contact is customer/vendor or not in Odoo13 those fields are removed. We have added them back same as like older version. We have added same technical name as in previous version. We keep customer rank and supplier rank as it is is user set/unset customer or vendor checkbox in backend customer rank and supplier rank will automatically set/unset so this will have any side effect in other flow

KEY FEATURES:

  • Easy To Use.
  • Added Is Customer, Is vendor checkbox Into The Partner Form For Identification.
  • User Can Manually Set/ Unset Customer and Vendor Checkbox.
  • Added Customer Filter In Sale Order.
  • Added Vendor Filter In Purchase Order.

CONFIGURATION

For This No configuration Required. Just Install this Module.

DETAILS

You can see Is Customer and Is Supplier checkbox in to the partner form view user can manually Set/Unset those checkbox. In default Odoo13 set unset customer or verdon is not possible.
